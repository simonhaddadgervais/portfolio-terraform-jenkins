Cloud Portfolio Website 

SOURCE CODE

A website (this one), hosted on AWS, for my portfolio. The interesting bit was to deploy everything using IaC and implementing CI/CD.

  • Lambda to run a function written in Python and containerized with Docker,   to count and returns the number of visits on my website
  • ECR to store the Docker container image
  • DynamoDB to store the data
  • API Gateway to display the visitors count on my website
  • Terraform as the IaC solution to deploy all the infrastructure 
  • Jenkins for CI/ CD to automate Test Build and Deployment
  • Git as VSC and Github to store the repository
  • Docker the containerize the function
  • AWS as the Cloud solution with :
  • S3 to host the web page
  • Cloudfront and ACM to serve HTTPS requests globally
  • Route 53 as DNS

Wordpress hosted on AWS

SOURCE CODE

A Wordpress website powered by Nginx on an Ubuntu EC2 instance

The purpose of this project was to get myself used to set-up a production grade webserver, on Linux, running one or multiple WordPress applications. I manually deployed and configured that infrastructure many times, on Linux, until I was familiarized enough with the process and then, I scripted all the deployment. Finally, I deployed everything using Ansible, leveraging its power and its simplicity.

  • Hosted on Linux EC2 instance.
  • SSH for remote configuration.
  • Configured Nginx, PHP, Wordpress, MySQL.
  • Monitored with Monit.
  • Used Ansible on a separate server for IaC deployment.
  • Alternatively, there is a Bash script version of the deployment
